🛠️ Projects

  • 🧠 VSAPI: A Visual Saliency API to analyze where people look. This API consists of over 20 predictive models, 8 evaluation metrics, and 10 datasets. This repo is currently private.
  • 🔍 VAE: A varietional auto encoder implemented in TensorFlow and PyTorch to learn to generate human faces that look realistic. The model trains on 10000 images only.
  • 🖼️ VQVAE_CelebA: A vector quantized varietional auto encoder implemented in PyTorch to learn to reconstruct human faces. This model is trained on the Celeb_A dataset.
  • 🔢 VQVAE_mnist: A generative model that learns digits representation via VQVAE, and digit generation using PixelCNN. This model is trained on the MNIST dataset.
  • 🌐 DCGAN: An implementation of a generative AI approach in TensorFlow and PyTorch to learn to generate fake human faces that look realistic. The model is robust and uses only 10000 images to learn to generate faces.
  • 🩺 Pneumonia Classification: A comprehensive evaluation of pneumonia classification using four models including custom CNN, transfer learning, partial fine-tuning, and full fine-tuning.
  • 🎮 Tetris: An implementation of the famous Tetris game.
